# Sumslack爬虫平台
## 简介
使用极少的脚本代码，自动将任意网站的数据爬到`MySQL`库或`MongDB`中，该爬虫程序是通用的，可根据您的需要量身定制。
## 脚本API文档
脚本API直达：[http://spider-api.sumslack.com](http://spider-api.sumslack.com)
